# System Resource Report

A professional Linux system project focused on generating system resource reports using Bash scripting.

---

## Overview

This project helps system administrators collect key system information such as CPU usage, memory usage, disk usage, uptime, and running processes.

---

## Features

- Display system uptime
- Check memory usage
- Check disk usage
- Show CPU information
- List top running processes
- Generate quick system health report

---

## Project Structure

- resource_report.sh → Main system report script
- commands.txt → Linux commands used
- README.md → Project documentation
