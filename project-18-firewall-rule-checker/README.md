# Firewall Rule Checker

A professional Linux system project focused on reviewing firewall status and active firewall rules using Bash scripting.

---

## Overview

This project helps system administrators check firewall configuration, review active rules, and verify basic system protection status.

---

## Features

- Check firewall status
- Display active firewall rules
- Support UFW firewall checks
- Show open ports
- Useful for security auditing
- Simple Bash automation

---

## Project Structure

- firewall_check.sh → Main firewall checking script
- commands.txt → Linux commands used
- README.md → Project documentation
