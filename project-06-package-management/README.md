# Package Management Lab

A Linux lab project focused on managing software packages using apt and dpkg.

---

## Overview

This project demonstrates how to install, update, remove, and manage packages in Linux systems using APT and DPKG tools.

---

## Features

- Update package lists
- Upgrade installed packages
- Install new packages
- Remove packages
- Search for packages
- Manage .deb files

---

## Project Structure

- commands.txt → Linux commands used in this lab
- README.md → Project documentation

---

## Commands Used

### Update package list
```bash
sudo apt update

----------------

Upgrade packages

sudo apt upgrade

Install a package

sudo apt install curl

Remove a package

sudo apt remove curl

Search for a package

apt search curl

Install a .deb package

sudo dpkg -i package.deb

Remove a .deb package

sudo dpkg -r package-name
