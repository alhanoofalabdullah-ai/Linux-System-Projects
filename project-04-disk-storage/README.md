# Disk & Storage Lab

A Linux lab project focused on disk usage, storage inspection, and mount-related commands.

---

## Overview

This project demonstrates how to inspect disk space, analyze directory sizes, and understand basic storage and mount operations in Linux.

---

## Features

- Check disk space usage
- Check directory size
- List block devices
- View mounted file systems
- Understand mount operations

---

## Project Structure

- commands.txt → Linux commands used in this lab
- README.md → Project documentation

---

## Commands Used

### Check disk space
```bash
df -h

-----------

Check directory size

du -sh .

Check size of folders

du -h --max-depth=1

List block devices

lsblk

Show mounted file systems

mount

Show a specific disk usage path

df -h /home

Check file system type

lsblk -f
