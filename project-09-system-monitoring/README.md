# System Monitoring Lab

A Linux lab project focused on monitoring system performance and resource usage.

---

## Overview

This project demonstrates how to monitor CPU usage, memory usage, system uptime, and active processes in Linux.

---

## Features

- Monitor running processes
- Check CPU and memory usage
- View system uptime
- Inspect disk usage
- Analyze active system load
