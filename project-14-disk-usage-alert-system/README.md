# Disk Usage Alert System

A professional Linux system project focused on monitoring disk usage and sending alerts when storage exceeds a defined threshold.

---

## Overview

This project helps system administrators monitor disk space usage automatically and detect storage issues before they become critical.

---

## Features

- Monitor disk usage percentage
- Set custom disk threshold alerts
- Display warning messages
- Simple automation using Bash
- Supports scheduled execution using cron

---

- disk_alert.sh → Main monitoring script

- README.md → Project documentation
