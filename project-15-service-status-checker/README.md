# Service Status Checker

A professional Linux system project focused on checking the status of system services using Bash scripting and systemctl.

---

## Overview

This project helps system administrators verify whether important Linux services are running, stopped, enabled, or disabled.

---

## Features

- Check service status
- Detect active and inactive services
- Display service availability
- Support multiple service names
- Useful for system health checks

---

## Project Structure

- service_check.sh → Main service checking script
- commands.txt → Linux commands used
- README.md → Project documentation

---

## Technologies Used

- Linux
- Bash Scripting
- systemctl
- Service Management
- System Monitoring
