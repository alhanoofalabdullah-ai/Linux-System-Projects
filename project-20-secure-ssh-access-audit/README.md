# Secure SSH Access Audit

A professional Linux system project focused on auditing SSH security settings and access configuration using Bash scripting.

---

## Overview

This project helps system administrators review SSH configuration, detect weak settings, and improve server access security.

---

## Features

- Check SSH service status
- Review SSH port configuration
- Verify root login settings
- Check password authentication status
- Basic SSH security audit
- Useful for server hardening

---

## Project Structure

- ssh_audit.sh → Main SSH audit script
- commands.txt → Linux commands used
- README.md → Project documentation
