# Network Connectivity Tester

A professional Linux system project focused on testing network connectivity using Bash scripting and common networking tools.

---

## Overview

This project helps system administrators verify network connectivity, DNS resolution, and basic reachability for important hosts.

---

## Features

- Test internet connectivity
- Ping multiple hosts
- Check DNS resolution
- Display reachable and unreachable hosts
- Useful for basic network troubleshooting

---

## Project Structure

- network_test.sh → Main network testing script
- commands.txt → Linux commands used
- README.md → Project documentation

---

## Technologies Used

- Linux
- Bash Scripting
- ping
- nslookup
- Networking
- Troubleshooting
