# Log Rotation Automation

A professional Linux system project focused on automating log cleanup and rotation using Bash scripting.

---

## Overview

This project helps manage system logs by automatically rotating old log files, compressing backups, and keeping the system clean and organized.

---

## Features

- Automatic log rotation
- Compress old logs using gzip
- Delete very old logs
- Backup important logs
- Improve disk space management
- Simple Bash automation

---

## Project Structure

- rotate_logs.sh → Main log rotation script
- logs/ → Sample log files
- archive/ → Compressed old logs
- commands.txt → Linux commands
- README.md → Project documentation
