# User Activity Audit Script

A professional Linux system project focused on auditing user activity, login history, and system access events using Bash commands.

---

## Overview

This project helps monitor user activity on a Linux system by checking login history, active users, failed login attempts, and recent authentication events.

---

## Features

- Check currently logged-in users
- Review user login history
- Display last login records
- Detect failed login attempts
- Audit authentication logs
- Support basic security monitoring

---

## Project Structure

- audit_users.sh → Main audit script
- commands.txt → Linux commands used
- README.md → Project documentation
