# Process Management Lab

A Linux lab project focused on managing system processes.

---

## Overview

This project demonstrates how to monitor, manage, and control processes in Linux.

---

## Features

- View running processes
- Monitor system usage
- Search for processes
- Kill processes
- Run background tasks

---

## Project Structure

- commands.txt → Linux commands used in this lab
- README.md → Project documentation

---

## Commands Used

### View running processes
```bash
ps aux

-------------

Monitor system processes

top

Search for a process

ps aux | grep firefox

Run a process in background

sleep 100 &

View background jobs

jobs

Bring job to foreground

fg

Kill a process by PID

kill 1234

Force kill a process

kill -9 1234
