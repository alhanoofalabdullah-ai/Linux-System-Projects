# Networking Basics Lab

A Linux lab project focused on basic networking commands and troubleshooting.

---

## Overview

This project demonstrates how to check network configuration, test connectivity, and inspect network interfaces in Linux.

---

## Features

- View IP address
- Check network interfaces
- Test internet connectivity
- Inspect routing table
- Check open ports

---

## Project Structure

- commands.txt → Linux commands used in this lab
- README.md → Project documentation

---
