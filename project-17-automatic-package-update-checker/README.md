# Automatic Package Update Checker

A professional Linux system project focused on checking available system package updates using Bash scripting.

---

## Overview

This project helps system administrators quickly identify available package updates and keep Linux servers secure and updated.

---

## Features

- Check available system updates
- Support Debian/Ubuntu systems
- Display upgradeable packages
- Improve server maintenance workflow
- Useful for regular system health checks

---

## Project Structure

- update_checker.sh → Main update checking script
- commands.txt → Linux commands used
- README.md → Project documentation

- apt
- Package Management
- System Maintenance
